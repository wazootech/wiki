# JavaScript

**JavaScript (JS)** is a programming language that is one of the core technologies of the World Wide Web, alongside [[html]] and [[css]].

JavaScript enables interactive web pages and is an essential part of web applications. Modern strongly-typed extensions include [[typescript]].

Data interchanged in web applications often uses the JSON format, which led to [[json-ld]] for linked data applications.
