# Notation3

**Notation3 (N3)** is a shorthand non-XML serialization of [RDF](RDF.md) models, designed with human-readability in mind.

N3 is much more expressive than Turtle; it provides syntax for RDF itself but also for rules, variables, and quoted graphs. [Turtle](Turtle.md) is a simplified subset of N3 that is focused purely on RDF statements.
