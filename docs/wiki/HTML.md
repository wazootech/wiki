# HTML

**HTML (HyperText Markup Language)** is the standard markup language for creating documents designed to be displayed in a web browser.

HTML provides the base structure for content, which is then styled with [CSS](CSS.md) and can support functionality via [JavaScript](JavaScript.md). Modern HTML supports embedding semantics natively through [Microdata](Microdata.md).
