# JavaScript

**JavaScript (JS)** is a programming language that is one of the core technologies of the World Wide Web, alongside [HTML](HTML.md) and [CSS](CSS.md).

JavaScript enables interactive web pages and is an essential part of web applications. Modern strongly-typed extensions include [TypeScript](TypeScript.md).

Data interchanged in web applications often uses the JSON format, which led to [JSON_LD](JSON_LD.md) for linked data applications.
