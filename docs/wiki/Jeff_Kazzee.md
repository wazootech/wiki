# Jeff Kazzee

[Jeff Kazzee](https://github.com/Jeff-Kazzee) builds zero-dependency tooling for **agent-native workspaces**: typed Markdown knowledge graphs, validation gates, self-improving agent loops, and graph-aware review by blast radius. His tools share a design law — the framework must cost almost nothing to load, or it steals the context the work needs.

## Software in this wiki

| --------------------------- | ----------------------------------------------------------------------------------- |

| [Loam](Loam.md)             | Folder-as-type typed knowledge layer; derives metadata from path, git, and H1       |
| [Vivary](Vivary.md)         | Agent workspace standard — tropo graph, strato loop, ozone review, exo coordination |

## Lineage
