# Turtle

**Turtle** (Terse RDF Triple Language) is a syntax and file format for expressing data in the [RDF](RDF.md) data model.

Turtle is very compact and readable by humans, which makes it the most common syntax used to manually write RDF data and axioms.

It is a subset of [Notation3](Notation3.md) and a superset of the minimal [N Triples](N_Triples.md) format.
