# Wiki CLI Agent Skills

[![skills.sh](https://skills.sh/b/wazootech/wiki)](https://skills.sh/wazootech/wiki)

Agent skills for [Wiki CLI](https://github.com/wazootech/wiki). They live under `skills/` and are **not** wiki content — do not add this folder to `wiki.inputs`.

Independent modules — no required order. Typical flow: **install** → **create** → **deploy** → **improve** (each optional).

| Skill | Purpose |
| ----- | ------- |
| [wiki-install](wiki-install/SKILL.md) | Install and verify the CLI |
| [wiki-create](wiki-create/SKILL.md) | `wiki init` plus preferences wizard |
| [wiki-improve](wiki-improve/SKILL.md) | Survey wiki hygiene; prioritized findings report |
| [wiki-deploy](wiki-deploy/SKILL.md) | GitHub Pages workflow and path alignment |

Docs: [Wiki_Skills.md](../docs/wiki/Wiki_Skills.md) and per-skill pages under [docs/wiki/](../docs/wiki/). User walkthrough: [Getting_Started.md](../docs/wiki/Getting_Started.md).
